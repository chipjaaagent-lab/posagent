# POSAgent

ระบบจัดการต้นทุน สูตร ออเดอร์ และกำไรสำหรับร้านอาหาร

## ฟีเจอร์

- 🏪 **รองรับหลายร้าน** — สร้างร้านได้หลายแห่ง ข้อมูลแยกกัน
- 🧾 **จัดการวัตถุดิบ** — คำนวณต้นทุนต่อหน่วยอัตโนมัติ
- 📝 **ตั้งสูตรเมนู** — ฟิกสูตรครั้งเดียว ใช้ตลอด
- 🛒 **เปิดออเดอร์** — รองรับช่องทางขาย GP/Ads/คูปองปรับได้
- 📊 **สรุปกำไรสุทธิ** — เห็นกำไรจริงหลังหักทุกค่าใช้จ่าย
- 📱 **PWA** — ใช้งานบนมือถือ แท็บเล็ต offline ได้
- 💾 **localStorage** — ข้อมูลเก็บในเครื่อง ปลอดภัย

## Setup

```bash
npm install
npm run dev
```

เปิด http://localhost:5173
